# Videos Folder

Esta carpeta contiene los videos que se pueden usar en la aplicación.

## Cómo añadir videos:

1. Coloca tus archivos de video (.mp4, .webm, .ogg, etc.) en esta carpeta
2. En el formulario de edición de jugador, escribe solo el nombre del archivo (ej: "Lebron.mp4")
3. La aplicación automáticamente detectará que es un archivo local y lo cargará desde aquí

## Formatos soportados:
- MP4 (.mp4)
- WebM (.webm)
- OGG (.ogg, .ogv)
- AVI (.avi)
- QuickTime (.mov)

## URLs externas:
También puedes usar URLs completas de YouTube o videos externos.

## Ejemplos:
- `Lebron.mp4` → se carga desde `/assets/videos/Lebron.mp4`
- `https://www.youtube.com/watch?v=dQw4w9WgXcQ` → video de YouTube
- `https://example.com/video.mp4` → video externo
